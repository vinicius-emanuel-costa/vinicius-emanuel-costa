### Sobre mim

Analista de Infraestrutura focado em **monitoramento, observabilidade e redes**. Experiencia com gestao de **59+ servidores**, redes mesh VPN multi-site com WireGuard, stacks completas de monitoramento e automacao de processos.

- Construo e mantenho stacks de observabilidade com **Prometheus, Grafana, Zabbix e OpenSearch**
- Projeto redes **mesh VPN com WireGuard** para ambientes de producao
- Desenvolvo ferramentas de **automacao e gestao de infraestrutura**
- Crio pipelines de **alerta inteligente** com correlacao de logs e IA

---

### Experiencia

**DevOps Engineer** — Planejamento e manutencao de infraestrutura cloud e on-premise, deploy de stacks de monitoramento, gestao de redes mesh VPN e automacao de processos de deploy.

**Suporte de Infraestrutura** — Administracao de servidores Linux, troubleshooting em producao, gerenciamento de 59+ hosts via SSH, manutencao de ambientes criticos.

**Desenvolvedor de Automacoes** — Criacao de ferramentas internas (bots, CLIs, plataformas web), scripts de automacao e pipelines CI/CD com GitHub Actions.

**Analise de Dados** — Desenvolvimento de solucoes para extracao, processamento e visualizacao de dados corporativos.
